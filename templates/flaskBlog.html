Installation ⬇️

download source code from Github 💾

git clone https://github.com/DogukanUrker/flaskBlog.git

go to directory 📁

cd flaskBlog

install requirements.txt 🔽

pip install -r requirements.txt

it's ready to run 🎉

python app.py

or

python desktop.py

for run flaskBlog as a desktop app 💯
