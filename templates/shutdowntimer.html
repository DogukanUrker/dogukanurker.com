Installation ⬇️

download source code from Github 💾

git clone https://github.com/DogukanUrker/ShutdownTimer.git

go to directory 📁

cd ShutdownTimer

install requirements.txt 🔽

pip install -r requirements.txt

it's ready to run 🎉

python ShutdownTimer.py
