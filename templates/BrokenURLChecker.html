Installation ⬇️

download source code from Github 💾

git clone https://github.com/DogukanUrker/BrokenURLChecker.git

go to directory 📁

cd BrokenURLChecker

Backend

cd backend

install requirements.txt 🔽

pip install -r requirements.txt

it's ready to run 🎉

python main.py

Frontend

cd frontend

install packages 📦

npm install

it's ready to run 🎉

npm run dev
