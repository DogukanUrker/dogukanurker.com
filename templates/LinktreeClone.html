Installation ⬇️

download source code from Github 💾

git clone https://github.com/DogukanUrker/LinktreeClone.git

go to directory 📁

cd LinktreeClone

Backend

cd backend

install requirements.txt 🔽

pip install -r requirements.txt

it's ready to run 🎉

uvicorn main:app --reload --port 1202

Frontend

cd frontend

install packages 📦

npm install

it's ready to run 🎉

npm run dev
